# 🍎 Sum 10 – 사과 퍼즐 (Pygame)

> 숫자가 적힌 사과들을 드래그해서 **합이 10이 되도록 선택**하세요!  
> 집중력 테스트와 순간 계산력을 한 번에 길러주는 미니게임 🎮

---

## 🎯 게임 설명

- 17 × 10 보드에 랜덤 숫자 (1 ~ 9) 사과들이 배치됩니다.
- 마우스로 **드래그 영역을 그려** 여러 사과를 선택합니다.
- 선택한 숫자들의 **합이 10이면 그 사과들이 제거되고 점수 + 1**
- 빈 칸은 그대로 유지되고 다시 채워지지 않습니다.
- **100초 제한 시간** 동안 최대한 많은 사과를 제거하세요!
- `다시 시작` 버튼으로 언제든 새 게임 시작 가능합니다.

---

## 🕹️ 조작 방법

| 동작               | 설명                                 |
| ------------------ | ------------------------------------ |
| **마우스 드래그**  | 선택 영역 그리기 (합 = 10 이면 제거) |
| **ESC**            | 게임 종료                            |
| **다시 시작 버튼** | 보드 초기화 및 점수 리셋             |
